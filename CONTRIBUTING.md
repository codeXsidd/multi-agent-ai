# Contributing to Multi AI Agent Platform

First off, thank you for considering contributing to Multi AI Agent Platform! It's people like you that make this open-source community such a great place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please create an issue utilizing the Bug Report template. Ensure to include detailed steps to reproduce, expected behavior, and actual behavior.

### Suggesting Enhancements
Feature requests are highly encouraged! Please use the Feature Request template to submit your idea. Explaining your use case gives the maintainers a better understanding of the value the feature would add.

### Pull Requests
1. Fork the repo and create your branch from `dev`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code respects the existing formatting/style.
5. Create a Pull Request using the provided PR template.

## Development Setup
Please refer to the setup instructions in the [README.md](README.md).

## Code of Conduct
Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
